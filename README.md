# Portfolio Website

- Portfolio website built with Astro and Tailwind CSS.
- email for feedback at contactayushmadhav@gmail.com

## Local Deployment

```bash
npm install
npm run dev
```

Open the local site in your browser:

- Actual portfolio: `http://localhost:4321/`
- UX sandbox copy: `http://localhost:4321/reference`

To test the production build locally:

```bash
npm run build
npm run preview
```
